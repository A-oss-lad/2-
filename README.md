# 江西粤东气象社区交流安全中心系统

## 项目简介

江西粤东气象社区交流安全中心系统是一个专为气象社区设计的安全管理平台，用于社区用户管理、违规举报处理、处罚通知及申诉等功能，包含用户与管理员两种登录角色，实现社区安全治理全流程线上化管理。

## 功能特点

### 用户端功能
- **登录/注册**：支持普通用户通过QQ号码和密码登录
- **举报功能**：用户可提交违规举报，包含被举报人信息、违规类型、描述和证据
- **处理进度查询**：用户可查看自己提交的举报记录及处理状态
- **处罚通知**：接收处罚通知，查看详情并提交申诉
- **个人中心**：查看个人信息、账号状态和历史记录

### 管理员端功能
- **管理员登录**：专用账号密码登录
- **控制台**：查看社区数据统计和最近活动
- **举报处理**：查看、处理用户提交的举报
- **申诉处理**：查看、处理用户提交的申诉
- **数据管理**：管理用户、处罚、规则等数据

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- 本地存储 (localStorage)

## 数据存储说明

本系统使用浏览器的 `localStorage` 存储数据，数据仅存储在本地浏览器中，刷新页面或更换浏览器会导致数据丢失。实际生产环境中应使用后端数据库存储数据。

## 注意事项

1. 本系统为前端演示系统，所有数据仅存储在浏览器本地
2. 系统中的图片均使用随机图片服务，实际应用中应使用真实图片
3. 系统中的时间计算基于客户端本地时间
4. 请不要在公共电脑上使用本系统，以免数据泄露
